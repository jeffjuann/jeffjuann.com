# Markdown Syntax Guide

## Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

## Emphasis

*Italic*  
_Italic_  

**Bold**  
__Bold__  

***Bold and Italic***  
___Bold and Italic___  

## Lists

### Unordered List

- Item 1
  - Subitem 1
  - Subitem 2
- Item 2
- Item 3

### Ordered List

1. First item
2. Second item
3. Third item

## Links

[Inline link](https://www.example.com)  

[Reference link][1]  

[1]: https://www.example.com "Optional Title Here"

## Images

## Blockquotes

> This is a blockquote.  
> This is the second line of the blockquote.

## Code

Inline `code`

```javascript
// Code block
function greet() {
  console.log("Hello, world!");
}
```
